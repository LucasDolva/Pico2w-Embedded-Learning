# Pico2w-Embedded-Learning
Firmware development and hardware interfacing on the Raspberry Pi Pico 2 W (RP2350) using MicroPython.

## 🚀 About This Repository
This repository serves as a hands-on laboratory journal for my embedded systems engineering journey. It documents my progress moving from basic digital input/output (GPIO) control to advanced sensors, hardware debouncing, and eventually automated systems.

## 🛠️ Hardware Stack
* **Microcontroller:** Raspberry Pi Pico 2 W (RP2350 ARM Cortex-M33 / Hazard3 RISC-V)
* **Development Environment:** Thonny IDE
* **Language:** MicroPython
