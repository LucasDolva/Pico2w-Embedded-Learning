# 04 - Button Debouncing (Enable/Disable Control)

This project implements a software-based **button debouncing** using MicroPython.


## 💡 Key Takeaway
* **Debouncing Filtering** Learning how to eliminate these false triggers using **time.sleep** function as a timing based filter to reliably toggle an LED state.
